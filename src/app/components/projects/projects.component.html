Synthboard

Synesthesia is condition in which a person experiences the stimulation of multiple senses when only one sense is evoked. In my case I see colours when I hear music. In this project I demonstrate the colours that I mostly see when the notes of c-major scale are played. This was implemented using HTML5 Canvas, animejs and Tonejs. It is named "Synthboard" because it is a blend of synesthesia and a keyboard.
